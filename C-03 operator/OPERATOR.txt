[3장] 연산자

(산술 연산자)

사칙연산 : + , -, *, /, **, %, // 

(비교 연산자)
GT : >        (크다)
GE : >=       (크거나 같다)
LT : <        (작다)
LE : <=       (작거나 같다)
EQ : ==       (같다)
NE : !=       (같지 않다)
결과 : bool

(논리 연산자)

and : 논리곱
or : 논리합
not : 부정
^ : 배타적 논리합(xor)
결과 : bool

(연산자 우선순위)

1. 리스트, 딕셔너리, 세트, 튜플  -> 컬렉션 언어
2. 거듭제곱 
3. 곱셈, 나눗셈, 정수나눗셈, 나머지
4. 덧셈, 뺄셈
5. 부정, 비교
6. 논리 연산자
7. 대입 연산자
